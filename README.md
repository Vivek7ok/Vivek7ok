# About Me

I’m a Data Analyst who’s proficient in using SQL, Python, Excel, and Power BI to analyze data in order to detect trends and solve problems faced by businesses.

The steps that I take when analyzing data include data cleaning, exploratory data analysis, use of SQL queries, creation of visualizations and dashboards. I also leverage artificial intelligence to automate tedious processes in order to increase efficiency and reduce the time it takes to conduct an analysis.

-  Passionate about data-driven problem solving
-  I make dashboards people *actually* want to look at
-  Heavy AI tools user — prompt engineering, LLM workflows, automation
-  Currently leveling up in cloud data with AWS
-  Open to collaborations, internships, and freelance analytics work
